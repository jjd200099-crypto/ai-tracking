# AI App Mobile Intelligence Dashboard

AI 应用移动端数据追踪仪表盘，实时监控 ChatGPT、Gemini、Claude 三大 AI 应用的下载量、收入、市场份额等关键指标。

## 功能

- 📊 三大 AI 应用（ChatGPT / Gemini / Claude）KPI 概览
- 📈 下载量与收入趋势图表（支持日/周/月粒度切换）
- 🌍 分国家/地区数据对比
- 📱 iOS & Android 平台数据
- 🔄 数据定期更新
